# Repositorio-bootcamp-github-pages
Criando site do portfólio
Neste repositório estamos desenvolvendo um site de portfólio pessoal e profissional como parte das atividades do Bootcamp I da graduação em Ciência da Computação. 
O projeto tem como objetivo aplicar, na prática, conhecimentos de desenvolvimento web, organização de projetos e utilização do GitHub Pages para publicação do site. 
Durante o desenvolvimento, serão criadas páginas para apresentação pessoal, habilidades, projetos, informações acadêmicas e formas de contato, contribuindo para a construção de uma presença digital profissional. 
Além disso, o projeto também auxilia no aprendizado de tecnologias e ferramentas utilizadas no desenvolvimento web moderno.
